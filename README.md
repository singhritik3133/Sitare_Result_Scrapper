# Sitare Result Scraper

This project automatically scrapes student result data (CGPA)
from the SITARE portal using Selenium and Python.

## Features
- Automatic login
- Fetches result PDF
- Extracts CGPA
- Saves data to Excel

## Requirements
- Python 3.8+
- Selenium
- Pandas

## How to Run
1. Install dependencies
2. Run main.py
